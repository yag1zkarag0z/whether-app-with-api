# Whether App with API

A simple weather application that provides real-time weather information for any city.

## Features

- Search weather by city name
- Displays temperature, humidity, wind speed, and weather conditions
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
